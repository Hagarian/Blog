Network Vulnerability Assessment

This project involved conducting a comprehensive vulnerability assessment on a simulated corporate network environment. I used Nmap for network discovery and service enumeration, identifying open ports and potentially vulnerable services. Nessus was then employed to perform in-depth vulnerability scanning, which revealed several critical vulnerabilities including outdated software versions and misconfigured services.

After identifying the vulnerabilities, I researched each one to understand the potential impact and likelihood of exploitation. I then categorized them based on risk level and provided detailed remediation strategies for each finding. The final report included executive summaries for management as well as technical details for IT staff.

Web Application Penetration Test

In this project, I performed a complete penetration test on a deliberately vulnerable web application (DVWA). The assessment covered all aspects of the OWASP Top 10, with particular focus on SQL Injection, Cross-Site Scripting (XSS), and Broken Authentication vulnerabilities.

Using Burp Suite as my primary tool, I intercepted and manipulated requests to identify vulnerabilities. For SQL Injection, I demonstrated how to extract database information including usernames and passwords. For XSS vulnerabilities, I created proof-of-concept payloads that could steal session cookies. The project also included recommendations for secure coding practices and web application firewall rules to mitigate the identified risks.

SIEM Implementation Lab

This project involved setting up a Security Information and Event Management (SIEM) system using Splunk in a lab environment. I configured data inputs to collect logs from various sources including Windows Event Logs, firewall logs, and Apache web server logs.

After establishing log collection, I created custom correlation searches to detect suspicious activities such as multiple failed login attempts, unusual outbound traffic patterns, and potential brute force attacks. I then developed alert rules to notify security personnel of these events in real-time. The project also included the creation of dashboards for visualizing security events and trends, helping to identify potential security incidents more efficiently.
